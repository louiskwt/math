# Derivative

$` f'(x) = lim_{x \to 0} \dfrac{f(x - h) + f(x)}{h}`$

$`f'(x) = \frac{dy}{dx}`$

## Common Derivatives and properties

- $`\frac{d}{dx}(k) = 0`$

- $`\frac{d}{dx}(x) = 1`$

- $`\frac{d}{dx}(x^n) = nx^(n-1)`$

- $`\frac{d}{dx}(sinx) = cosx`$

- $`\frac{d}{dx}(cosx) = -sinx`$

- $`\frac{d}{dx}(tanx) = \frac{1}{cos^2x}`$

- $`\frac{d}{dx}(e^x) = e^x`$

- $`\frac{d}{dx}(lnx) = \dfrac{1}{x}`$

- $`\frac{d}{dx} = \frac{1}{dy/dx}`$
